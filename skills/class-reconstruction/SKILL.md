# Class Reconstruction

Take whatever a teacher remembers about a class — pose fragments, a peak posture, a
playlist, a one-line theme — and rebuild it into a complete, usable sequence they can
re-teach, share, or archive.

## When to use this

- A teacher improvised a class that worked and wants to capture it before it's gone.
- A sub needs to recreate a class on short notice from someone's rough notes.
- A teacher has a theme and a peak pose but needs the full arc filled in.
- Building a personal library of reusable sequences.

## Inputs to gather

Ask for whatever exists — none of these are required, more is better:

- **Style & length** — e.g. Vinyasa 60 min, Yin 75 min, Hatha 45 min.
- **Level** — all-levels, beginner, intermediate, advanced.
- **Theme or intention** — the thread of the class (physical, energetic, or philosophical).
- **Peak pose / focus** — the apex the class builds toward, or the body area of focus.
- **Fragments the teacher remembers** — any poses, transitions, cues, or moments.
- **Playlist** — song order often mirrors the energetic arc of the class.
- **Props / constraints** — blocks, bolsters, straps, wall; room or population limits.

## Process

1. **Collect** every fragment the teacher offers without judging the order yet.
2. **Find the arc.** Identify the peak and the theme, then work backward: what has to be
   warm and open to reach the peak safely?
3. **Build the sequence** through the standard arc, scaled to the style and length:
   - **Centering / arrival** — breath, intention-setting, theme introduction.
   - **Warm-up** — gentle mobilization of the joints and tissues the peak needs.
   - **Build** — sun salutations or progressive standing work to generate heat.
   - **Peak** — the apex pose or sequence, with clear preparation.
   - **Cool-down** — counter-poses, twists, forward folds to neutralize.
   - **Savasana / close** — rest, and a callback to the theme.
4. **Sequence safely.** Honor counter-posing and warm-up logic; flag anything that asks the
   body for the peak before it's prepared.
5. **Add teaching support** where useful: a sentence of theme language, 1–2 cues per key
   pose, and obvious modifications/variations.
6. **Deliver** the reconstructed class and note any gaps you filled by inference so the
   teacher can correct them.

## Output format

```markdown
# [Class title or theme] — [Style], [Length], [Level]
**Theme:** [one or two sentences]
**Peak:** [pose or focus]
**Props:** [list, or "none"]

## Sequence
| Phase | Pose / action | Cue or note | Breath / count |
| --- | --- | --- | --- |
| Centering | ... | ... | ... |
| Warm-up | ... | ... | ... |
| Build | ... | ... | ... |
| Peak prep | ... | ... | ... |
| Peak | ... | ... | ... |
| Cool-down | ... | ... | ... |
| Close | Savasana | ... | ... |

## Theme thread
[Opening line · mid-class callback · closing line]

## Notes
- Inferred / filled in by reconstruction: [list — confirm these]
- Modifications: [...]
```

## Style notes

Adjust the arc to the tradition — don't force a Vinyasa shape onto every class:

- **Vinyasa / Power** — breath-linked flow, clear heat build to a peak.
- **Hatha** — longer holds, less linking, more precision per pose.
- **Yin** — few poses, long passive holds (3–5+ min), targets connective tissue; "peak"
  is depth and stillness, not intensity.
- **Restorative** — fully supported shapes, deep rest, minimal effort.
- **Ashtanga** — fixed series; reconstruction means identifying *where in the series* and
  noting deviations rather than inventing a sequence.

## Guardrails

- **Safety first.** Never sequence a strong peak without adequate warm-up; respect
  counter-posing. Flag contraindications for common conditions when relevant.
- **Name the inferences.** Be explicit about what you filled in versus what the teacher
  actually remembered, so they stay the author of their own class.
- **Use real pose names.** Offer Sanskrit and English together where it helps
  (e.g. *Utkatasana* / Chair).
